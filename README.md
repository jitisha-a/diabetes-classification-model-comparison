# diabetes-classification-model-comparison

## About
Comparison of KNN, Decision Tree, and Logistic Regression classifiers for predicting diabetes status using survey data in R.

## Dataset
This project uses the [Diabetes Prediction Dataset by Mohammed Mustafa on Kaggle](https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset), containing 100,000 survey responses for diabetes status prediction.
